# -A-HYBRID-CYPTOGRAPHIC-ARCHITECTURE-FOR-SECURE-CLOUD-FILE-SHARING-WITH-BLOCKCHAIN-AND-ZKP
A hybrid cryptographic system for secure cloud file sharing combining AES-256-GCM encryption, RSA-2048 key exchange, Zero-Knowledge Proofs (Schnorr protocol), Proxy Re-Encryption, and Ethereum blockchain auditing — backed by AWS S3 and Cognito.
